# 💍 Wedding Outcome Estimator

An interactive web application that estimates potential wedding outcomes based on user inputs, built with React, TypeScript, and Vite.

---

## 📌 Overview

The **Wedding Outcome Estimator** is a frontend-focused project that combines user input processing with custom logic to generate dynamic and engaging results.

Inspired by culturally relevant themes, the application provides a fun yet structured way to simulate compatibility and outcomes through an intuitive interface.

---

## ✨ Features

* 🎯 Interactive user input system (age, GPA/CGPA, etc.)
* ⚡ Real-time result calculation
* 🎨 Premium UI with smooth user experience
* 📊 Custom logic-based outcome estimation
* 🏆 Bangladesh-style themed result cards
* 🖨️ Print-friendly result view
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** React
* **Language:** TypeScript
* **Build Tool:** Vite
* **Architecture:** Component-based UI

---

## 🚀 Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

Then open:

```text
http://localhost:5173
```
